# AGENTS Split: Output Rules

## Output Policy
- `RULE-OUT-001` 出力は最小構成（変更点 / 実行結果 / 未解決事項）を既定とする。
- `RULE-OUT-002` 成功時は冗長なログや差分全文を出さず、必要最小限のみ報告する。
- `RULE-OUT-003` 失敗時は `失敗点` `原因候補` `再実行方針` の3点を優先して短く報告する。
- `RULE-OUT-004` 仮定と `unknown` は最終報告に明示する。
- `RULE-OUT-005` 追加詳細（全コマンドログ、詳細差分、拡張説明）はユーザー明示要求時のみ展開する。
- `RULE-OUT-006` 実装結果報告では、実行した検証コマンドを1〜2行で記載する。
