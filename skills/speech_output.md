# Speech Output Skill

## 目的
- 発話用の通知ファイルを生成する
- 実装作業の有無に関係なく利用できる

## 起動条件
- `skills/end_phase.md` から呼び出されたとき
- 発話通知のみを単独で生成したいとき

## 出力ファイル

出力先:
C:\Users\gonec\.codex\runtime/agent_event_end.md

保存:
- 常に上書き
- UTF-8 の LF 改行で保存する
- 末尾改行あり
- 3行固定
- 自然文のみ
- 見出しや箇条書きや記号を使わない
- ファイル名やパスやコマンド名を書かない

内容:
- 1行目は今回の作業の要約を書く
- 2行目は実施した主な対応を書く
- 3行目は確認結果と終了状態を書く

## 失敗時

- ルート `runtime/agent_event_end.md` への書き込み失敗時はルート `runtime` ディレクトリ作成を1回試行したうえで同一ファイルへ1回再試行する
- 再試行でも失敗した場合は処理を中断し失敗理由を対話で明示する

## 禁止事項

- runtime 以外への保存
- 推測による補完
