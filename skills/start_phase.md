# Start Phase Skill

## 目的

受信した要求の理解と、その理解に基づく次の行動を、
読み上げ専用ファイルへ上書き出力する。

このフェーズではコマンド実行を行わない。

## 起動条件

- 実装依頼を受けたとき
- `AGENTS.md` または `skills/*.md` の変更依頼を受けたとき
- 本フェーズ完了後は Implementation Phase を実行する

## 出力先

ルート runtime/agent_event_start.md

## 出力方式

- 上書き保存
- UTF-8 の LF 改行で保存する
- 2行固定
- 自然文のみ
- 記号や見出しや箇条書きを使わない

## 失敗時

- ルート `runtime/agent_event_start.md` への書き込みに失敗した場合はルート `runtime` ディレクトリ作成を1回試行したうえで再書き込みする
- 再書き込みでも失敗した場合は処理を中断し、失敗理由を対話で明示する

## 出力内容

本文は2行で構成する。

1行目は今回の要求をどう理解したかを書く。
2行目はその理解に基づき次に何を行うかを書く。
