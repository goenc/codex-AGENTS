# UIフォントスキル

目的:
  同梱フォントを標準UIフォントとして使用する

必須条件:
  - フォントファイルが以下に存在すること
      assets/fonts/NotoSansJP-Regular.ttf
  - ライセンスファイルが以下に存在すること
      assets/fonts/OFL.txt
      assets/fonts/FONT_SOURCE.txt
  - ライセンスファイルは削除してはならない

実行時ルール:
  - フォントは assets/fonts/NotoSansJP-Regular.ttf からのみ読み込む
  - OSフォントを使用しない
  - フォールバックを行わない
  - フォント読み込みに失敗した場合は明確なエラーを出して終了する

統合ルール:
  - 既存のUIフォント登録処理に同梱フォントを渡す
  - プロジェクトに既に存在しないフレームワーク固有処理は追加しない
  - 無関係なUIコードを変更しない

完了確認:
  - 日本語が正しく表示されること（例: 「あア漢字」）
